Installation Guide

  1. Download the Installer

    Click the download button for your operating system above.

  2. Run the Installer

    Windows: Run the .exe file and follow the setup wizard.

    macOS: Open the .dmg file and drag Neura Desktop to Applications.

    Linux: Make the AppImage executable: chmod +x Neura-Desktop.AppImage

  3. Launch the Application

    Start Neura Desktop from your applications menu or desktop shortcut.

  4. Import Your .neura File

    Use the "Import Profile" feature to load your Neura Profile created on the web platform.
